# Installation

## Install Package

```bash
pip install paymcp mcp
```

## Development Install

```bash
git clone https://github.com/yourusername/paymcp.git
cd paymcp
pip install -e .
```

## Requirements

- Python 3.10+
- MCP SDK (`mcp` package)

## Optional

For webview support (desktop):
```bash
pip install paymcp[webview]
```
